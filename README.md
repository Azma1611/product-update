# Turmeric Product Comparator

This is a simple web tool to:

- Paste product data (Perundurai / Erode Market / ERODE SOCIETY)
- Separate the data by place
- Compare Finger & Bulb prices with previous data
- Show ↑ (green) if price increased, ↓ (red) if decreased
- Store data in browser for future comparisons

## How to use

1. Clone the repository or upload it to GitHub.
2. Open `index.html` in your browser (or enable GitHub Pages).
3. Paste product data into the textarea and click "Save & Compare".


# Turmeric Product Comparator

## How to use

1. Clone the repository or upload it to GitHub.
2. Open `index.html` in your browser (or enable GitHub Pages).
3. Paste product data into the textarea and click "Save & Compare".
4. Data is automatically separated by place, prices are compared with previous data, and arrows indicate price changes.
